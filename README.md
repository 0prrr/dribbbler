# Dribbler

实现让平面设计师方便地从dribble.com下载图片的功能；设计师只需要输入想要搜索的关键词即可默认下载10页的相关图片到特定目录；

## 已不再维护

## Version Log

V0.1

1.用户输入一个关键词，下载10页相关图片到特定文件夹；
2.能标记处文件存放的位置；
3.能标记出总共下载的文件个数；
4.支持png，jpg和gif格式的下载；

V1.0

1.处理下载链接，将最后下滑点到后缀名部分去除，下载分辨率最高的图片；
2.增加文档注释

V1.1

1.增加批量下载，用户可以输入多个关键字；
2.增加异常处理，异常时自动重试并提示用户；
3.提升效率，过滤重复图片url；
